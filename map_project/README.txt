# Neighborhood map project 
This is a map example for my account om udacity
(https://classroom.udacity.com/nanodegrees/nd001/syllabus)
 **Front End Web Developer Nanodegree**.

*** application describtion *****
This is a single-page application featuring a map of schools in nasr city and new cairo in egypt
Application works with wikipedia and Google Map APIand knockoutjs framework.

*****  to open the app: ******
1- right click on index.html to run it on browser
2- enter school name that you want to search
3- the marker show it's location 
4- to know more about it click to marker to show you it's infowindow

*****  features:  ****
1- google map
2- markers 
3- info window
4- search box
